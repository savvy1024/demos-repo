## This is a simple demo created by React FrontEnd and Node/Express BackEnd.

Client is the FrontEnd

api is the BackEnd

## How to run

## api

-1. cd api
-2. npm install
-3. npm start

## client

-1. cd client
-2. npm install
-3. npm start

##Check
When the two apps running, open brower in http://localhost:3000/, you will see the data fetch from the api.
